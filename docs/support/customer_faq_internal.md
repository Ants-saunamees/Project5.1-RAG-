# Internal Customer FAQ Reference

This document contains internal explanations for common customer questions. It is not visible to customers but helps support agents provide accurate answers.

## 1. Why does the app require location access?
The app uses location data to personalize content and improve recommendations. Location is optional, and users may disable it at any time.

## 2. Why do some features require a subscription?
Premium features require additional server resources and ongoing maintenance. Subscriptions fund these costs and allow continued development.

## 3. Why do updates take time to roll out?
Updates are released gradually to ensure stability. If issues occur, the rollout can be paused.

## 4. Why does the app log certain events?
Event logging helps diagnose issues and improve performance. Sensitive data is never logged.

## 5. Why do users get logged out sometimes?
Session tokens expire for security reasons. Users may also be logged out when switching devices.

This FAQ helps support agents provide clear and consistent explanations.
